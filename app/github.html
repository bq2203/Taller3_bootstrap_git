Guía GitHub ADSO — Resumen rápido

Paso 1 — Verificar la instalacion del git
  • git --version
Paso 2 — Configurar tu identidad
  • git config --global user.name "Tu Nombre"
  • git config --global user.email "tu.email@example.com"
Paso 3 — Inicializar proyecto local
  • cd ruta/a/tu/proyecto
  • git init
Paso 4 — Agregar archivos y commit
  • git add .
  • git commit -m "first commit"
Paso 5 — Crear un repositorio en GitHub
  • Inicia sesion en GitHub
  • Haz clic en "New repository"
  • Ingresa un nombre para el repositorio
  • Selecciona "Initialize this repository with a README"
  • Haz clic en "Create repository"
Paso 6 — Conectar el proyecto local con el repositorio en GitHub
  • git remote add origin https://github.com/tuusuario/nombre-del-repositorio.git
  • git branch -M main
  • git push -u origin main
Paso 7 — Publicar proyecto
  • git push
