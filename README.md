# Taller3_bootstrap_git
# Práctica de Git y GitHub

## Descripción

Este proyecto fue desarrollado como práctica para aprender el uso de Git y GitHub, aplicando conceptos como la creación de repositorios, control de versiones, commits y sincronización de proyectos locales con repositorios remotos.

Además, el proyecto incluye diferentes páginas web desarrolladas con HTML y Bootstrap, entre las que se encuentran formularios de registro e inicio de sesión, ejemplos de formularios, diseño con columnas y una página informativa relacionada con el Mundial de Fútbol.

## Funcionalidades

* Página principal (Home).
* Registro de usuarios.
* Inicio de sesión.
* Página sobre el Mundial de Fútbol.
* Ejemplos de formularios con Bootstrap.
* Diseño de columnas utilizando el sistema Grid de Bootstrap.
* Organización de archivos mediante carpetas para estilos, imágenes y scripts.

## Tecnologías Utilizadas

* HTML5
* CSS3
* Bootstrap 5
* Git
* GitHub

## Objetivos de Aprendizaje

* Comprender el funcionamiento de Git.
* Gestionar versiones de un proyecto web.
* Utilizar GitHub como repositorio remoto.
* Aplicar Bootstrap para crear interfaces responsivas.
* Organizar adecuadamente la estructura de archivos de un proyecto web.

## Cómo Ejecutar el Proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/usuario/repositorio.git
```

2. Abrir la carpeta del proyecto.

3. Ejecutar el archivo `index.html` en cualquier navegador web.
